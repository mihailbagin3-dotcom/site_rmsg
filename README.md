# RUMESSAGE -  НОВАЯ СОЦ.СЕТЬ ДЛЯ СТРИМЕРОВ!
Добро пожаловать в RUMESSAGE — уникальную социальную сеть, созданную специально для сообщества стримеров и их зрителей!

Что вы найдёте здесь:

эксклюзивные посты от стримеров: анонсы стримов, закулисные фото, личные новости и многое другое;

возможность следить за несколькими авторами в единой ленте;

прямой доступ к страницам стримеров без регистрации.

Как найти любимого стримера? Очень просто! В адресной строке вашего браузера наберите: ник_стримера.ru-message.ru. Например, если ник стримера — StreamMaster, введите StreamMaster.ru-message.ru, и вы мгновенно попадёте на его профиль.

Присоединяйтесь к RUMESSAGE и будьте на связи с миром стриминга!
